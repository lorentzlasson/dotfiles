# dotfiles
Collection of my personal dotfiles

## Setup
1. `cd ~`
1. `git clone https://github.com/lorentzlasson/dotfiles`
1. `cd dotfiles`
1. `rm ~/.config` 
1. `rm -rf ~/.config && mkdir -p ~/.config && stow .`
